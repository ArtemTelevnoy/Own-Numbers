# Own-Numbers
Find own numbers of square matrix(Not always, realisation isn't very good)

running:
```
gcc main.c -o main
./main inputFileName outputFileName
```

input file format:
```
n
a11 a12 ... a1n
a21 a22 ... a2n
...
an1 an2 ... ann
```
where n-size of square matrix, aij-element of matrix on [i][j] position
